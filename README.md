# SR Diffusion Model as a Data Augmentation Method in CNN
The code used in my Master's Thesis. I tested how a super-resolution diffusion model performs as a data augmentation method when a CNN is trained with data generated by the diffusion model and compared to CNNs that were trained with high- and low-resolution data.
